This extension integrates Burp with the Dradis Framework.

Dradis is a collaboration and reporting tool for IT Security teams that lets you combine the output of 20 different scanners with your own manual findings and screenshots to create completely custom reports.

After you load the extension you can right-click on the Issues identified by Burp's Scanner and send them to Dradis. This extension uses Dradis JSON HTTP API interface.

Dradis editions supported by this connector: Dradis Community and Dradis Professional.

Usage:

  • For Dradis Community edition use the server's shared password as API token.
  • For Dradis Professional edition find your API token in your Profile page.
